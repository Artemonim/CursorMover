# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-03

### Added
- Initial release
- `copy` command to copy workspace folders with chat history
- `move` command to move workspace folders with chat history
- `doctor` command to inspect workspace ID mapping
- `merge` command to consolidate duplicate workspaceStorage entries
- Interactive TUI for all operations
- Database lock checking for safety
- Workspace storage merge capability
- Automatic venv management via bootstrap
- Cross-platform support (Windows, macOS, Linux)
- Comprehensive test suite
- CLI with full argument parsing
